# AirCurrencyWeb

## 环境配置

```
    1.环境隔离
        pip3 install pipenv

    2.启动配置
        vim ~/.bashrc
        export FLASK_ENV='production'

    3.安装依赖
        cd /srv/AirCurrencyWeb
        pipenv install
    
    4.不带web容器启动
        cd /srv/AirCurrencyWeb
        pipenv shell
        pipenv python3 run.py
    
    5.web容器+Nginx启动


```
